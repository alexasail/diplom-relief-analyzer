В статье рассмотрена разработка телеграм-бота для автоматизированного анализа цифровых моделей рельефа (DEM) неспециалистами без установки ГИС-систем. Проведен обзор пяти open-source инструментов: GRASS, WhiteboxTools, OpenTopography, QGIS, xDEM — где WhiteboxTools выделяется максимальным набором топографических индексов, а QGIS лидирует по активности сообщества. Разработана архитектура ReliefAnalyzer, которая загружает высокоточные DEM-данные из OpenTopography, применяет медианную фильтрацию для устранения шумов, вычисляет уклоны методом Хорна и формирует наглядную визуализацию в Matplotlib с картами высот и уклонов. Бот на базе aiogram обеспечивает простой интерфейс: пользователь отправляет координаты и мгновенно получает готовое PNG-изображение с легендами и подписями. Docker-контейнер с локальным кэшем гарантирует кросс-платформенный запуск без дополнительных настроек. Решение снижает порог входа для анализа рельефа, сочетая точность профессиональных инструментов с доступностью мессенджера.
